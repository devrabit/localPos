# 🧠 CONTEXTO DEL PROYECTO POS

---

## 🏗️ Stack

- Frontend: Vue 3 + Pinia + Tailwind
- Backend: Node.js + Express
- DB: MySQL
- Integración: WooCommerce API

---

## 📦 Funcionalidades actuales

- Ventas
- Productos
- Variaciones de productos
- Escaneo de código de barras
- Métodos de pago:
  - efectivo
  - tarjeta
- Cálculo de cambio
- Historial de ventas

---

## 📌 Reglas importantes

- Soporte para productos con variaciones
- POS optimizado para mobile
- UI dinámica basada en specs
- Sincronización con WooCommerce

---

## 🧩 Arquitectura

- Modular
- Basada en specs (.md)
- Separación frontend/backend
- Escalable
